Foundation-runtime.github.io

Foundation Runtime is a set of modules that we designed at Cisco for server applications. We thought they could be helpful to the broader community so we open sourced them. The modules stand by themselves, but are most helpful when used together to provide a holistic solution to standard needs of enterprise server systems. The mail modules are:

  • Common Configuration
  • Communication API - this includes HTTP client and Server, Async Messaging
  • Logging (including structured logging)
  • Service Discovery infrastructure
  • Monitoring
  • Other goodies

  • Smart defaults
  • Client side load balancing
  • Structured logging
  • Automatic correlation ids in log files to assist tracking a single transaction from different processes' logs
  • Async Messaging
  • Service Directory
  • Central Configuration
